Help

blink-suyb is the cross-platform, browser-UI edition of SMACK UP YOUR BACKUP. It is the same engines — nothing is a re-implementation.

  • Where's my data? blink-suyb auto-detects your portable SUYB folder and reads its config, profiles and credentials as-is.
  • Why does it open in Chrome? The UI is HTML served from a localhost server. The backup work happens in the local Python process, so FTP/SFTP/disk/scheduling all work.
  • Is it secure? The server binds to 127.0.0.1 only and requires a per-launch token.
